Turing Machine

Non-Deterministic Finite Automaton

Inserte Texto

Pagina para hacer la conversion de texto a cadenas binarias con la ayuda de una maquina de turing previamente desarrollada.

Inserte una Operacion

Pagina para hacer la validacion de una cadena, la cual representa una serie de operaciones, si las operaciones estan correctamente planteadas la cadena se aceptara.

Binary Text

A continuacion se muestran las cadenas resultantes al realizar la conversion de texto a binario.

Operation

Si la operacion es valida, se retornara el resultado de dicha operacion, si no, se mostrara un mensaje rechazando la cadena.
